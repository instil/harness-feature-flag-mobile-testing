◆ InitiatePlaybackOf() [1/3]

void SurgeXamariniOSBindings.SurgeRtspPlayer.InitiatePlaybackOf ( NSUrl  url)

Initiate playback of the RTSP stream.

This will issue the required DESCRIBE, SETUP and PLAY RTSP requests to start stream playback. Once completed, Surge will notify the user of the requests result via the Surge RTSP delegate. If successful, the video stream will start playing on the provided ImageView associated with the SurgeRtspPlayer::PlayerView property.
